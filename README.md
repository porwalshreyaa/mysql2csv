# mysql2csv

Converts MySQL dump files (.sql) to CSV format using Bash.

## Installation

Clone the repository:
```bash
git clone https://github.com/yourusername/mysql2csv.git
cd mysql2csv
```

## Usage
```bash
./mysql2csv dump_file.sql > output.csv
```
